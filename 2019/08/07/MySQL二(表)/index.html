语法

主键约束

primary key [默认值]

PRIMARY KEY[字段一]

多字段联合主键

PRIMARY KEY[字段一,字段二]

外键约束

CONSTRAINT [外键名字] FOREIGN KEY(列) REFERENCES tb_name(列)

非空约束

唯一性约束

字段名 数据类型 UNIQUE

CONSTRAINT <约束名> UNIQUE(<字段名>)

默认约束

字段名称 类型 DEFAULT 默认值

设置自增

AUTO_INCREMENT

查看表结构

查看表基本结构

DESCRIBE 表名 简写：DESC 表名

查看表详细结构

SHOW CREATE TABLE <表名\G>

修改表

修改表名

ALTER TABLE <旧表名> RENAME <新表名>;

修改字段数据类型

ALTER TABLE <表名> MODIFY <字段名> <数据类型>;

修改字段名

ALTER TABLE <表名> CHANGE <旧字段名> <新字段名> <新数据类型>; 数据类型不能为空

添加字段

ALTER TABLE <表名> ADD <新字段名> <数据类型> [约束条件] [FIRST] [ATFER 已存在字段名]

删除字段

ALTER TABLE <表名> DROP <字段名>

修改字段排序

ALTER TABLE <表名> MODIFY <字段1> <数据类型> AFTER <字段2>

更改表的存储引擎

ALTER TABLE <表名> ENGINE=<更改后的存储引擎名>

删除表的外键约束

ALTER TABLE <表名> DROP FOREIGN KEY <外键名>

删除表

DROP TABLE <表名>
